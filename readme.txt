# Glamorous Look - Jewelry and Accessories Website

## Overview
This is a complete e-commerce website for Glamorous Look, a jewelry and accessories brand. The website includes product browsing, shopping cart functionality, checkout process, user account management, and informational pages.

## Website Structure

### Main Pages
- Home Page (index.html): Landing page with featured products and collections
- Single Product Page (single-product.html): Detailed product view with gallery, description, and reviews
- Shopping Cart (cart.html): View and manage items in cart
- Checkout (checkout.html): Complete purchase process
- My Account (my-account.html): User account management
- About Us (about-us.html): Company information
- Contact (contact.html): Contact information and form

### Information Pages
- FAQs (faqs.html): Frequently asked questions
- Returns & Refunds (returns-refunds.html): Return policy information
- Safety & Security (safety-security.html): Security information
- Privacy Policy (privacy-policy.html): Privacy policy details
- Error Page (error-page.html): 404 error page

### Blog Pages
- Blog Classic (blog-classic.html): Main blog page
- Blog 4 Column (blog-4-column.html): Alternative blog layout
- Blog with Sidebar (blog-with-sidebar.html): Blog with sidebar navigation
- Single Post (single-post.html): Individual blog post view

### Additional Pages
- Order Tracking (order-tracking.html): Track order status
- Wishlist (wishlist.html): User's saved items

## Technical Information

### File Structure
- HTML files: Main website pages
- CSS folder: Contains styling files including Bootstrap and custom styles
- JS folder: JavaScript files for functionality
- Images folder: All website images and assets

### Dependencies
- Bootstrap: For responsive layout and components
- jQuery: JavaScript library for DOM manipulation
- Swiper: For image sliders and carousels
- AOS: For scroll animations

## Features
- Responsive design for all device sizes
- Product filtering and sorting
- User account management
- Shopping cart and checkout process
- Product reviews and ratings
- Related products suggestions
- Newsletter subscription
- Contact form
